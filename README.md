# Pemesanan-tiket-mrt
Aplikasi pemesanan tiket MRT yang berbasis Java dan mengimplementasikan konsep OOP. Pengguna dapat memilih stasiun asal dan tujuan, menentukan jumlah penumpang, serta memilih kelas gerbong. Sistem menghitung harga secara otomatis berdasarkan jarak dan kelas yang dipilih. Pembayaran digital dikirim langsung ke akun pengguna.
